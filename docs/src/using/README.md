# Using Litmus

Litmus is a web app for previewing terminal color themes. This section walks through its main features.

- **[Browsing Themes](./browsing.md)** — find themes by name, variant, or readability
- **[Theme Detail](./detail.md)** — inspect a single theme across all fixtures
- **[Comparing Themes](./comparing.md)** — side-by-side comparison with contrast analysis
- **[Exporting Config](./exporting.md)** — generate config files for your terminal
